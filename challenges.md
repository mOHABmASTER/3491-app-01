# Git Basic Tasks

## Task 1

* Create a GitHub account
* Create a new folder called `git-practice`
* Initialize a Git repository inside it
* Create a file called `index.html`
* Add a basic HTML structure to `index.html`

---

## Task 2

* Create a file called `about.html`
* Add a basic HTML structure
* Create your **first commit** with a proper message

---

## Task 3

* Update `about.html`
* Add a paragraph describing the website
* Create a **second commit**

---

## Task 4

* Create a file called `contact.html`
* Add a simple contact section (title and paragraph)
* Create a **third commit**

---

## Task 5

* Delete the file `about.html`
* Create a **final commit**
* Check the commit history

---

## Task 6

* Create a new repository on GitHub
* Connect your local project to the GitHub repository
* Push your project to GitHub

---

## Task 7

* Update `index.html` and add a new section
* Create a new commit with a clear message
* Push the new changes to GitHub
